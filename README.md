# House Price Prediction (Machine Learning)

## Overview
A regression project to predict house prices based on property features, comparing multiple machine learning models to identify the best performer.

## Key Results
- **Best Model:** Random Forest Regressor
- **Performance:** R² = 0.77 on test data

## Tools Used
- Python
- Pandas, NumPy (data handling)
- Scikit-learn (modeling)
- Matplotlib/Seaborn (visualization)

## Process
1. **Data Cleaning & EDA** — Explored the dataset to understand feature distributions, identify missing values, and detect outliers
2. **Feature Engineering** — Created and transformed features to improve model performance
3. **Model Training** — Trained and compared multiple regression models (e.g., Linear Regression, Decision Tree, Random Forest)
4. **Model Evaluation** — Compared models using R² and error metrics; Random Forest performed best with R² = 0.77
5. **Insights** — Identified which features had the strongest influence on predicted house prices

## Files
- `house_price_prediction.ipynb` — Full notebook with EDA, feature engineering, model training, and evaluation

## Dataset
california house price dataset
